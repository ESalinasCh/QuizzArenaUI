# QuizzArenaUI

Repositorio del cliente web para la plataforma QuizzArena, construido con **Angular 17+** y **Tailwind CSS**.

> ⚙️ **Nota:** Este repositorio contiene exclusivamente la interfaz de usuario (Frontend). Para consultar el código del servidor (.NET 8), la base de datos o el motor de Inteligencia Artificial, dirígete al repositorio Core: [**QuizzArena**](https://github.com/ESalinasCh/QuizzArena).
